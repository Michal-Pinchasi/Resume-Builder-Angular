# Smart CV Builder - Angular Reactive System

מערכת דינמית ליצירת קורות חיים המבוססת על Angular, המאפשרת הזנת נתונים וצפייה בתוצאה בזמן אמת.

## אודות הפרויקט
הפליקציה מאפשרת בניית קורות חיים בצורה אינטראקטיבית תוך דגש על ולידציות מחמירות, ניהול טפסים מורכבים ותצוגה מקדימה מיידית (Preview) של התוצר הסופי.

## טכנולוגיות
- Angular 17+ (Standalone Components)
- Reactive Forms & FormBuilder
- FormArray לניהול רשימות דינמיות
- TypeScript & CSS3

## פיצ'רים מרכזיים
- **פרטים אישיים:** אימות פורמט אימייל וחסימת תווים שאינם ספרות בשדה הטלפון.
- **ניהול השכלה:** אפשרות להוספת מספר מוסדות לימוד, כאשר כל מוסד כולל רשימת קורסים פנימית.
- **לוגיקה מותנית:** הסתרת שדה תאריך סיום וביטול חובת מילוי כאשר הסטטוס הוא "לומדת כרגע".
- **בקרת תאריכים וציונים:** ולידטור מותאם אישית המוודא שתאריך הסיום מאוחר מתאריך ההתחלה, ובדיקת טווח ציונים (0-100).
- **תצוגה מקדימה:** סנכרון מלא בין שלב הזנת הנתונים למסמך הסופי המוצג למשתמש.

## מבנה נתונים
- PersonalDetails: ניהול פרטי הפרופיל.
- EducationItem: מודל מוסד לימודים הכולל סטטוס ותאריכים.
- CoursesArray: מערך קורסים דינמי המנוהל בתוך כל ישות השכלה.
